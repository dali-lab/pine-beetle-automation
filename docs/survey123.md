# `/survey123` routes

## `GET /upload`

- `ObjectID`
- `GlobalID`
- `Southern Pine Beetle`
- `State`
- `County/Parish`
- `Year`
- `Season`
- `Traps set out on:`

- `Person setting the trap (your name)`
- `Trap Lure`
- `Date of Inital bloom`
- `What bloomed?`
